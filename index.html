Blog

(Blog)

This is a full-featured blog application built using Appwrite as the backend for authentication, database management, and file storage. Users can sign up and log in securely using Appwrite's authentication system. Once logged in, they can create, edit, and delete blog posts, each consisting of a title, content, and optional image uploads. All blog data is stored in Appwrite's database, and images are handled via Appwrite's storage service, allowing for efficient file management and previewing.

WorldAtlas

(Country-Project)

This is a fully frontend-based Country Information App that allows users to browse and search for countries around the world. It fetches real-time data from a public API (like REST Countries) and displays key information such as the country name, flag, capital, population, region, and native language. Users can filter countries by region or search by name. Clicking on a country opens a detailed view with more in-depth information.
